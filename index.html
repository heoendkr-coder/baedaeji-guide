구매대행을 시작하면 제일 먼저 부딪히는 벽이 배대지입니다.

"배대지가 뭐야?" "어디 써야 해?" "주문서는 어떻게 작성해?"

이 가이드 하나로 배대지의 모든 것을 정리했습니다.

1. 배대지란 무엇인가

배대지 = 배송대행지

중국에서 상품을 사면, 한국으로 바로 보낼 수 없습니다. 타오바오 판매자가 한국 주소로 직배송? 안 됩니다.

배대지는 중국에 있는 '중간 물류 창고'입니다. 판매자가 배대지로 상품을 보내면, 배대지에서 검수하고, 포장하고, 한국까지 배송해줍니다.

배대지가 해주는 일 5가지

단계역할
① 입고판매자가 보낸 상품을 대신 수령
② 검수상품 상태를 확인하고 사진 촬영
③ 포장국제배송에 맞게 재포장 (파손 방지)
④ 출고세관 통관 서류 처리 + 한국행 선적
⑤ 국내배송통관 완료 후 택배사 통해 고객 배송

전체 배송 흐름

타오바오/1688 주문
    ↓ 판매자 발송 (2~4일)
배대지 (중국 산둥성 웨이하이)
    ↓ 검수 + 포장 (1~2일)
해운 선적 (3~4일)
    ↓
한국 세관 통관 (1~7일)
    ↓
소형 → CJ/한진 일반택배
대형 → 경동택배 화물
    ↓ (1~2일)
고객 수령

총 소요: 약 9~14일 (공휴일 제외)

2. 배대지 선택의 기술

배대지 선택 기준 4가지

기준왜 중요한가
C/S (소통)문제 생겼을 때 정확하고 빠르게 답변하는지. 이게 1순위
포장 품질국제배송 중 파손되면 끝. 포장이 허술한 곳은 쓰면 안 됩니다
통관 처리 능력가격취하, 합산과세 등 통관 이슈 대응 능력
배송비같은 무게에 얼마인지. 단, 싸다고 다 좋은 건 아닙니다

"좋고 나쁨보다, 나에게 맞느냐가 중요합니다."

넥스트배송을 추천하는 이유

  1. 포장이 꼼꼼합니다 — 국제배송 중 파손 위험 최소화
  2. 경동택배 연계 — 크무비(큰 상품) 하려면 경동택배 필수
  3. C/S가 정확합니다 — 카카오톡 채널톡으로 양식 보내면 처리
  4. 보험 시스템이 체계적 — 화물보험 + 배송비보험
  5. 샵마인 연동 — 자동으로 고객 정보 기입

처음 시작하는 분은 넥스트배송 하나만 쓰면 됩니다.

3. 넥스트배송 시작하기

3-1. 회원가입 & 초기 세팅

넥스트배송 사이트에서 회원가입하세요. (네이버 로그인 비추천, 일반 가입 권장)

가입 후 채널톡으로 "탄생님 수강생"이라고 말씀하시면 등급 적용해줍니다.

필수 세팅 3가지

  1. 운송방식 변경: 설정 > 계정관리 > 운송방식 인천2(해운) 선택
  2. SMS 수신 '예': 입고/출고 알림톡 수신
  3. 예치금 자동결제: 자동결제 기능 활성화 권장

3-2. 서비스 구분

서비스설명누가 쓰나
배송대행내가 타오바오에서 직접 사고, 배대지는 배송만기본. 이걸 씁니다
구매대행배대지가 대신 사고 + 배송까지수수료 5% 추가
사업자 LCL 선적사업자 전용 대량 해상운송물량 많아지면 고려

6. 트래킹번호 & 판매자 소통

6-1. 트래킹번호 입력 주의사항

주의사항설명
영문 포함 시 영문부터 입력숫자만 넣으면 입고 누락
실제 출고 후에만 발급판매자가 "보냈다"고 해도 실제 출고 전이면 번호 없음
2개 이상일 수 있음크기 크거나 수량 많으면 분할 발송
타오바오에서 1개만 보일 수 있음나머지는 알리왕왕으로 판매자에게 직접 확인

6-2. 판매자에게 반드시 확인할 것

① 포장 후 무게

중문: 打包后有多重？请告诉我打包后的公斤重量。
번역: 포장 후 킬로그램 무게를 알려주세요.

주의: 중국은 "근(斤)"을 쓰는데, 1근 = 약 0.5kg입니다. 반드시 KG 단위로 확인.

② 포장 상태 & 나무포장 여부

중문: 包装是什么样的？给我发一下图片吧。
중문: 是木箱包装么？
중문: 里面有缓冲包装么？

③ 총 박스 수 & 트래킹번호

중문: 总共几个箱子？每个箱子的快递单号是多少？
번역: 총 몇 박스인가요? 각 박스의 운송장번호는?

대형 상품은 소포 2개로 나뉘어 발송되는 경우가 있습니다. 반드시 확인하세요.

6-3. 멀티트래킹번호

하나의 주문서에 여러 트래킹번호를 입력할 수 있는 기능입니다. 소포 2개로 나오는 경우나 부품 재발급 시 사용합니다.

7. 입고부터 출고까지

7-1. 입고 소요 시간

구간소요 기간
판매자 → 배대지빠르면 2일, 보통 3~4일, 길면 7일+
입고 → 검수/포장1~2일

7-2. 임의서명 주의

중국 배대지 근무시간은 08:00~17:00. 저녁/이른 아침의 "서명 완료"는 택배기사의 임의 서명입니다.

  • 서명 후 1~2일 뒤에 실제 전달되는 경우 흔합니다
  • 서명 완료 후 2일 내 입고 안 되면 배대지에 문의

7-3. 입고 후 출고 프로세스

  1. 알림톡으로 입고 알림 수신
  2. 넥스트 홈페이지에서 "실사보기" 클릭 → 사진 확인
  3. 파손/이상 없으면 → 출고요청
  4. 문제 있으면 → 배대지 채널톡으로 소통

통관 중에는 주소변경이 매우 어렵습니다. 출고 전에 변경하세요.

8. 검수의 모든 것

8-1. 4가지 검수 유형

유형내용비용언제 쓰나
검수안함포장 그대로 출고무료빠른 출고 필요 시
기본검수외관/옵션/컬러 확인. 실사 1장무료기본적으로 이걸 씁니다
정밀검수제품 꺼내서 상세 촬영약 4,000~7,000원부품 확인 필요, 고가 상품
작동검수전원 ON/OFF만 확인유료전자제품 (전원코드 필수)

정밀검수 주의: 넥스트에서 자동 결제하지 않음 → 직접 결제하기 클릭 필수.

작동검수 한계: 전원 ON/OFF만 확인. 기능 테스트 불가. 조립 후 작동검수 불가. 전원코드 없으면 불가.

8-2. 검수 선택 가이드

상품추천 검수
일반 소형 상품기본검수
고가 전자제품정밀검수
부품 많은 상품정밀검수
가구/대형기본검수 + 필요 시 정밀
의류검수안함 or 기본검수
조립 제품/수량 3개 이상정밀검수 권장
밀봉 제품정밀검수 (개봉 확인 필요 시)

8-3. 기본검수 상세 규칙

  • 외관 위주 검수 + 옵션/컬러 확인
  • 무료 실사 1장 (1회 한정) — 외부 박스 또는 내부 박스 한 면만 촬영
  • 수량 3개 이상, 조립 제품, 밀봉 제품은 기본검수로 확인 어려움 → 정밀검수 필요
  • 기본검수 후 추가 사진 요청 시 → 정밀검수 비용 발생

8-4. 정밀검수 상세

  • 제품을 꺼내서 상세 촬영
  • 비용: 입고 시 박스 기준으로 산정
  • 노데이터 검수 요청 시 → 정밀검수로 진행
  • 포장완료 후 검수 요청 시 → 정밀검수로 진행
  • 넥스트에서 자동 결제하지 않음 — "결제대기"에서 직접 결제 클릭 필수

9. 포장 전략 — 크무비 특화

9-1. 포장 재료 종류

재료설명용도
에어스틱두꺼운 공기기둥기본. 잘 모르겠으면 에어스틱
에어캡 (뽁뽁이)버블랩저위험 소형 물품
각대 (모서리보호대)모서리 보호캡가구 필수
발포지완충 폼무거운 상품 (에어캡은 무게에 터짐)
추천포장배대지가 알아서처음엔 이걸로 + 사진 보내며 조율

9-2. 카테고리별 포장 기준

상품포장 방법보험
가구/책상발포지 + 추천포장 + 모서리보호대 + 에어스틱필수
기계류에어캡 또는 발포지 + 추천포장필수
유리 제품에어캡 + 발포지필수
일반 부품에어스틱권장
의류/직물포장 불필요불필요

9-3. 추천포장 상세

  • 에어캡/에어팩, 발포지, 우드 포장 등 상품 특성에 맞춰 배대지가 판단하여 적용
  • 파손 100% 방지 서비스는 아닙니다 — 최선의 포장이지 완벽한 보장은 아님
  • 목재 프레임 상품: 별도 요청 없으면 해체/내부 검수 미진행
  • 동일 제품이라도 작업자에 따라 포장 방식이 다를 수 있음
  • 처음엔 배대지에 상품 사진 보내고 추천 포장 방식 상의하는 것을 권장

9-4. 나무포장 함정

나무 종류수출 가능특징
원목 (가공 안 된 나무)불가거칠고 투박함. 통관 시 벌금 + 반송 가능
합판/압축목재가능매끈하고 가공된 느낌

판단 어려우면 판매자에게 포장 사진 받아서 배대지에 확인 요청.

9-5. 우드포장(나무포장) 요청

특별히 튼튼한 포장이 필요한 경우 우드포장을 요청할 수 있습니다.

  • 추가 비용 발생

16. 김탄생의 핵심 꿀팁

실전에서 꼭 알아야 할 것들

1. 주문서 복사 시 주소 변경 반드시 확인 — 이전 고객 주소가 남아서 오배송 사고 발생.

2. 정밀검수 결제는 수동으로 — "결제대기"에 쌓이면 직접 하나하나 결제해야 합니다.

3. 경동 송장은 오후 1~3시에 확인 — 확인 후 바로 마켓에 입력.

4. 소포 분리 배송 미리 고지 — 대형 상품은 2박스로 나뉘어 배송될 수 있음.

5. 경동택배비는 상품가에 포함 — 크무비는 경동비까지 합산해서 판매가 설정.

6. 타오바오 주문번호도 주문서에 기입 — 트래킹 확인이 원활해집니다.

7. 실사보기로 입고 상태 확인 — 파손/스크래치 확인 후 출고.

8. 통관 중 주소변경 거의 불가능 — 통관 전에 변경하세요.

9. 예치금 자동결제 활용 — 매번 수동 결제 = 시간 낭비.

10. 배대지는 첫 번째 사업 파트너 — 소통이 되는 곳, 문제를 함께 해결해주는 곳이 좋은 배대지입니다.
